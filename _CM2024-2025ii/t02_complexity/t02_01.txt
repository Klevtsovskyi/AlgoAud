f(n) = 3 n^2 - n + 4

a) f(n) = O(n^2)

3 n^2 - n + 4 <= C n^2
C = 3
3 n^2 - n + 4 <= 3 n^2
- n + 4 <= 0
4 <= n
n0 = 4
Для всіх n >= n0: 3 n^2 - n + 4 <= C n^2
Отже, f(n) = O(n^2)

b) f = Omega(n^2)

3 n^2 - n + 4 >= C n^2
C = 2
3 n^2 - n + 4 >= 2 n^2
n^2 - n + 4 >= 0
n (n - 1) + 4 >= 0
n0 = 1
Для всіх n >= n0: 3 n^2 - n + 4 >= C n^2
Отже, f(n) = Omega(n^2)
